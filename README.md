# skillbridge-AI
SkillBridge AI is an AI-powered platform that connects job seekers with relevant job opportunities by analyzing resumes, identifying skills, and recommending personalized learning paths.
# SkillBridge AI 🚀

AI-powered platform to:
- Analyze resumes
- Extract skills
- Suggest jobs
- Recommend learning paths

## Tech Stack
- React
- Node.js
- OpenAI API

## Run Project

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
